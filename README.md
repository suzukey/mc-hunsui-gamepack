# MC-Hunsui GamePack

ゲーム詰め合わせマインクラフトデータパック

minecraft version: 1.21.6

## 内容

- Liars: Liar's Bar のマインクラフト版
- PlayOut: オリジナルルールのトランプゲーム

## コマンド

### 導入

```minecraft
/reload

/datapack enable file/mc-hunsui-gamepack "/
```

## ゲーム

```minecraft
/function sui:select/
```
